
# VVRichText 

## 关于 

* VVRichText 是一个富文本显示,编辑组件，支持异步文本渲染。
* 可以在文本中插入本地图片，网络图片或UIView对象。
* 支持WebP、GIF等格式。
* 支持添加单击和长按事件到文本。
* 绘制文本框架，绘制空心字，设置文本垂直对齐属性等。
* 易于解析文本中的表达式，如http（s）链接，@user，＃theme＃，电话号码。
* 快速设置图像角半径属性和模糊处理。
